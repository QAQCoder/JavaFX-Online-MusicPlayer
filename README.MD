## Javafx在线音乐播放器

2019年毕业设计，网络音乐资源来自酷狗音乐，本项目仅供学习使用

学习过程中，参考了一些优秀的JavaFX项目：

1.JavaFX单机版音乐播放器：https://github.com/Mpmart08/MusicPlayer

2.JavaFX仿微信即时通讯客户端：https://github.com/GZYangKui/ChangLiao-Desktop
